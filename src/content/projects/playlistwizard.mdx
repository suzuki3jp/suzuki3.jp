## 概要

PlaylistWizard は、YouTube、YouTube Music、Spotify のプレイリストを直感的に管理・整理できる Web アプリケーションです。

プレイリストのコピー、シャッフル、マージ、抽出など、音楽ストリーミングサービス公式UIでは難しい操作を簡単に行えます。

## 主な機能

- **プレイリスト管理**: コピー、シャッフル、マージ、抽出、削除
- **Undo 機能**: プレイリスト操作を安全に取り消し
- **プレイリストブラウザ**: プレイリスト内の曲を検索・閲覧
- **プレイリストインポート**: 他のユーザーのプレイリストをインポート
- **マルチプラットフォーム**: YouTube、YouTube Music、Spotify に対応
- **多言語対応**: 英語・日本語

## 技術スタック

- YouTube Data API v3
- Spotify API
- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- TanStack Query
- NextAuth.js
- Jotai
- pnpm Workspace によるモノレポ構成
